# 생수가 흐르는 교회 찬양집
> 이 프로젝트는 생수가 흐르는 교회 찬양집 프로젝트입니다.

## Setup & Run
```
1. git clone https://github.com/livingwaterchurch0508/living-water-worship.git
2. yarn
3. yarn dev
```

## Specification
```
node : 20.9.0
next.js : 14.1.3
react : ^18
```

## Dependencies
```
@chakra-ui/react : 2.8.2 // UI 라이브러리 
zustand : 4.5.2 // 상태관리
```

## Deploy
```
yarn build
yarn start
```
